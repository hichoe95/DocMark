## [1.0.0] - 2026-02-15

### Added
- Initial release
- Project folder scanning for markdown files
- Beautiful markdown rendering with syntax highlighting
- Multi-project library with favorites and pins
- Full-text search across documents
- Quick Open for fast file navigation
- Mermaid diagram rendering
- KaTeX math equation rendering
- GitHub-style admonition support
- Table of Contents panel
- Git branch and status display
- Dark mode support
- Keyboard shortcuts for navigation
- `.docsconfig.yaml` support for project structure
- AI agent skills for Claude Code and OpenCode
